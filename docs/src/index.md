# biobench

Benchmark for biology-related computer vision tasks.

## Installation

```sh
curl -LsSf https://astral.sh/uv/install.sh | sh
uv pip install biobench
```

## Quick Start

Download a dataset (NeWT is easy to start with):

```sh
uv run biobench/newt/download.py --dir ./newt
```
